⚙️ Kaip aktyvuoti (OneSignal)
  1. Registruokitės: onesignal.com
  2. Sukurkite naują App → Web Push → įveskite savo domeno adresą
  3. Nukopijuokite App ID ir REST API Key
  4. Railway → Settings → Variables → pridėkite:
    ONESIGNAL_APP_ID=xxxx
    ONESIGNAL_REST_API_KEY=xxxx
  5. Redeploy → push notifikacijos iš karto veikia!
